# Mouse vs Cheese — 強化学習ラボ

ネズミ（Agent）が**方策勾配法（Policy Gradient）**で学習し、グリッドワールド上のチーズ（Reward）を見つけるシミュレーションです。

## 制作背景

### 課題
強化学習は近年のAI分野で重要な技術だが、数式やアルゴリズムの理解だけでは直感的に掴みにくい。教科書的な説明では「エージェントが報酬を最大化するように方策を学習する」と書かれていても、実際に学習率や割引率を変えると挙動がどう変わるのかは体験しないとわからない。

### 目的
- **強化学習の学習過程を可視化し、直感的に理解できる教材を作る**
- ハイパーパラメータの変更が学習にどう影響するかを、リアルタイムに実験・観察できるようにする
- ブラウザだけで動作するWeb版を提供し、環境構築なしで誰でも試せるようにする
- 方策勾配法の原理を「ネズミがチーズを探す」というわかりやすい題材で体験的に学べるようにする

## 技術選定

| 技術 | 選定理由 |
|:---|:---|
| **Python + pygame**（ローカル版） | 強化学習の実装で広く使われるPythonを採用。pygameによるリアルタイム描画でエージェントの動きを直感的に確認できる |
| **NumPy** | 行列演算・勾配計算を効率的に行うため。フレームワーク（PyTorchなど）を使わずNumPyでネットワークをスクラッチ実装することで、方策勾配法の内部動作を学習者が理解しやすくした |
| **Vanilla JavaScript**（Web版） | 外部ライブラリ・ビルドツールなしの単一HTMLファイルで完結させ、GitHub Pagesで即デプロイ可能にするため。Canvas APIで描画、Float64Arrayで行列演算を実装 |
| **方策勾配法（REINFORCE）** | 方策ベースの強化学習の基本手法であり、「行動確率を直接学習する」という仕組みが可視化に適している。Q学習と比べて方策の変化を確率分布として表示しやすい |
| **RMSProp** | SGDより収束が安定しつつ、Adamほど複雑でない最適化手法。学習者にとって理解しやすいバランスを重視 |
| **GitHub Pages** | 静的HTMLのみで構成されるため、サーバー不要で無料ホスティングが可能。URLを共有するだけで誰でもすぐに試せる |

## 仕組み

### 環境
- 15x15のグリッドワールド。壁（移動不可）と通路（移動可能）で構成
- チーズを取ると次の位置に移動（4箇所を巡回）

### 学習アルゴリズム
- **方策勾配法（REINFORCE）**: ニューラルネットワークが各方向（上下左右）の行動確率を出力
- **ネットワーク**: 入力層（225） → 隠れ層（ReLU） → 出力層（Softmax, 4方向）
- **最適化**: RMSProp

### 報酬設計
| 条件 | 報酬 |
|:---|:---|
| チーズに到達 | +1 |
| チーズに近づいた | +1 |
| チーズから遠ざかった | -1 |

### ハイパーパラメータ（デフォルト値）

| パラメータ | 値 | 説明 |
|:---|:---|:---|
| 学習率 | 0.0025 | パラメータの更新幅 |
| 割引率 γ | 0.98 | 未来の報酬の重視度 |
| 探索率 ε | 0.03 | ランダム行動の確率 |
| バッチサイズ | 4 | まとめて更新するエピソード数 |
| 隠れ層サイズ | 200 | 中間層のニューロン数 |
