# Mouse vs Cheese — 強化学習ラボ

ネズミ（Agent）が**方策勾配法（Policy Gradient）**で学習し、グリッドワールド上のチーズ（Reward）を見つけるシミュレーションです。

## デモ

**Web版（GitHub Pages）**: https://liofval.github.io/mousecheesegame/

ブラウザだけで動作します。インストール不要。

- ハイパーパラメータ（学習率・割引率・探索率など）をスライダーでリアルタイムに調整可能
- プリセット（ターボ学習・安定重視・大脳モデルなど）をワンクリックで切り替え
- ヒートマップ・方策表示・軌跡表示による学習過程の可視化
- 報酬推移グラフと行動確率分布のリアルタイム表示

**原理解説ページ**: https://liofval.github.io/mousecheesegame/docs.html

強化学習の仕組み、ネットワーク構造、報酬設計、各ハイパーパラメータの影響を解説しています。

## 仕組み

### 環境
- 15x15のグリッドワールド。壁（移動不可）と通路（移動可能）で構成
- チーズを取ると次の位置に移動（4箇所を巡回）

### 学習アルゴリズム
- **方策勾配法（REINFORCE）**: ニューラルネットワークが各方向（上下左右）の行動確率を出力
- **ネットワーク**: 入力層（225） → 隠れ層（ReLU） → 出力層（Softmax, 4方向）
- **最適化**: RMSProp

### 報酬設計
| 条件 | 報酬 |
|:---|:---|
| チーズに到達 | +1 |
| チーズに近づいた | +1 |
| チーズから遠ざかった | -1 |

### ハイパーパラメータ（デフォルト値）

| パラメータ | 値 | 説明 |
|:---|:---|:---|
| 学習率 | 0.0025 | パラメータの更新幅 |
| 割引率 γ | 0.98 | 未来の報酬の重視度 |
| 探索率 ε | 0.03 | ランダム行動の確率 |
| バッチサイズ | 4 | まとめて更新するエピソード数 |
| 隠れ層サイズ | 200 | 中間層のニューロン数 |
