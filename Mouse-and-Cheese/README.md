# Mouse-and-Cheese

## 概要

本プロジェクトは、強化学習の一種である\*\*Policy Gradient (方策勾配法)\*\*を用いて、ネズミ（Agent）がグリッドワールド環境でチーズ（Reward）を見つけることを学習するシミュレーションです。環境の視覚化には`pygame`を使用しています。

-----

## 特徴

  * **環境:** 15x15のグリッドワールド環境。壁（移動不可）と通路（移動可能）が存在します。
  * **学習アルゴリズム:** Policy Gradient法（REINFORCEアルゴリズムに類似）を採用しています。
  * **報酬設計:**
      * チーズに到達すると報酬 (+1) が与えられ、チーズの位置が更新されます。
      * さらに、エピソードのステップごとに、**チーズへの近接度に基づく報酬**が追加されます。前ステップよりもチーズに近づいた場合は報酬が加算され、遠ざかった場合は減算されます。
  * **ニューラルネットワーク:** 全結合層とReLU活性化関数で構成され、出力層には`SoftmaxWithLoss`を使用して行動確率を算出します。
  * **最適化:** RMSPropオプティマイザを使用しています。

-----

## 実行方法

### 必要なライブラリ

本プロジェクトを実行するには、以下のPythonライブラリが必要です。

  * `pygame`
  * `numpy`
  * `pickle`

これらのライブラリは、通常`pip`でインストールできます。

```bash
pip install pygame numpy
```

### 実行コマンド

`Mouse-and-Cheese/`ディレクトリのルートで、以下のコマンドを実行します。

```bash
python Mouse-and-Cheese/main.py
```

実行すると、`pygame`ウィンドウが開き、ネズミが学習しながら移動する様子が視覚化されます。コンソールにはエピソードごとの報酬の合計と移動平均が表示されます。

## ファイル構成

| ファイル/ディレクトリ | 役割 |
| :--- | :--- |
| `main.py` | メインの実行ファイル。環境の初期化、学習ループ、行動決定、報酬の記録、ネットワークの更新を行います。 |
| `environment/grid_world.py` | グリッドワールドの環境（`grid_world`クラス）を定義。描画、ステップ実行、報酬判定などを管理します。 |
| `environment/policy_gradient.py` | 強化学習エージェント（`network`クラス）を定義。ニューラルネットワーク構造、順伝播、逆伝播、報酬の割引計算、オプティマイザ（`RMSProp`）などが含まれます。 |
| `images/` | ゲームに使用される画像アセット（ネズミ、チーズ、背景チップ）を格納しています。 |

-----

## 詳細な学習設定

`environment/policy_gradient.py`で設定されている主なハイパーパラメータは以下の通りです。

| パラメータ | 値 | 説明 |
| :--- | :--- | :--- |
| `gamma` | `0.98` | 報酬の割引率。 |
| `batch_size` | `4` | 方策を更新するために収集するエピソード数。 |
| `epsilon` | `0.03` | ε-greedy法に基づく行動選択のε（ランダム行動の確率）。 |
| `lr` | `0.0025` | RMSPropオプティマイザの学習率。 |
| `decay_rate` | `0.97` | RMSPropの減衰率。 |
